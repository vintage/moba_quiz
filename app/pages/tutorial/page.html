Player requirements In order to successfully solve all type of questions you have to be familiar with the {{ sourceName }} game. All of that is because this quiz contains questions which are tightly connected with the world of {{ sourceName }}. You can learn more at:
Highscore Online leaderboard contains scores of best players divided into four categories: daily, weekly, monthly and overall. The most important part is that only single best score of the player will be shown in each of the categories to prevent score flooding.
Game modes During playing the game you will have to answer different type of questions. Providing valid answers gives you points which are essential to put your score at the top of the highscore table. Here are the type of questions which you have to be aware of:

Skill hero

guess which hero is using this ability

Hero name

choose the name for the hero

Hero skills

pick all skills which can be used by this hero

Attack type

choose if hero is using melee or range type of base attack

Item price

pick the right price which is needed to buy this item

Item recipe

remind yourself all components which needs to be used to craft the item
